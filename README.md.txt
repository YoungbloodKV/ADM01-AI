# AI Lab Experiments (Python)

This repo contains Python implementations of classic AI problems experiments given in curriculum:

1. 8 Puzzle Problem (A* Search)
2. 8 Queens Problem (Backtracking)
3. Water Jug Problem (BFS)
4. Crypt-Arithmetic Problem (Constraint Satisfaction)
5. Missionaries and Cannibals Problem (State-Space Search)
6. Vacuum Cleaner Problem (Reflex Agent)
7. Breadth-First Search (BFS)
8. Depth-First Search (DFS)
9. Travelling Salesman Problem (Brute Force)
10. A* Algorithm (Graph Search)
